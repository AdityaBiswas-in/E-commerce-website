## ✨ Features

### 🛒 Add to Cart
- Add any product to the shopping cart with a single click
- Update product quantity directly from the cart
- Remove individual items or clear the entire cart
- Real-time cart item count displayed in the navbar
- Persistent cart data using **localStorage**

### ❤️ Wishlist
- Save favorite products to a personal Wishlist
- Toggle items in/out of the Wishlist with a heart icon
- Move items from Wishlist directly to the Cart
- Wishlist data stored in **localStorage** for persistence

### 💳 Payment & Checkout
- Checkout form with billing details
- Multiple payment options:
  - 💳 Credit / Debit Card
  - 📱 UPI Payment
  - 💵 Cash on Delivery (COD)
- Order confirmation page after successful payment
- Form validation for secure and complete input

### 👤 User Account System
- Sign Up and Sign In pages
- Personal account page to manage profile
- Order history tracking via Orders page

### 🏠 General
- Attractive landing page with featured products and banners
- Product detail page for individual items
- Collections page to browse by category
- About page with brand/store information
- Fully **Responsive Design** — works on mobile, tablet, and desktop
- Clean and modern UI with smooth interactions

## 🔄 How It Works

### 🛒 Cart Flow
```
index.html → product.html → Add to Cart → cart.html → checkout.html → confirmation.html
```

### ❤️ Wishlist Flow
```
product.html / collections.html → Click ♡ → wishlist.html → Move to Cart
```

### 👤 Auth Flow
```
signup.html → signin.html → account.html → orders.html
```

### 💳 Payment Flow
```
cart.html → checkout.html → Choose Payment Method → confirmation.html ✅
```
